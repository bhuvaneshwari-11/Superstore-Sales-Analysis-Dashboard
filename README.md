# Superstore Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance using the Superstore dataset.

## Tools Used
- Power BI
- Excel Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold KPI
- Sales by Category
- Sales by Region
- Sales Trend by Order Date
- Sales by Sub-Category
- Sales by Ship Mode
- Profit by Category
- Top 10 Products by Sales

## Key Insights
- Total Sales: 2.33M
- Total Profit: 292.30K
- Technology category generated the highest sales and profit.
- West region contributed the highest sales.
- Standard Class was the most used shipping mode.
